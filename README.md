# UIDAI Data Hackathon 2026  
## Age-wise Aadhaar Enrolment Analysis

### Overview
This project analyses age-wise Aadhaar enrolment data provided by UIDAI
to understand which age groups contribute most to total enrolments.

### Tools & Technologies
- Python
- Pandas
- Matplotlib

### Dataset
- Aadhaar Enrolment Dataset (age_0_5, age_5_17, age_18_greater)

### Analysis Performed
- Aggregated enrolment by age group
- Percentage contribution calculation
- Bar chart visualisation

### How to Run
```bash
python analysis.py
